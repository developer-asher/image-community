# SNS Practice in React

## 기능 및 API 설계

- session storeage, cookie, firebase auth를 사용한 로그인, 로그아웃
- firestore를 이용한 데이터 관리
- fire storage를 이용한 업로드 파일 관리
- loadash를 이용한 잦은 이벤트 관리(무한스크롤)
- query를 이용한 데이터 sorting
